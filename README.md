## lomeXpress - Dashboard Admin

Système de gestion d'inventaire et vitrine e-commerce pour articles solaires et produits importés.

### Fonctionnalités
- **Dashboard Privé** : Gestion du stock, ajout de produits et modification des prix.
- **Vitrine Client** : Interface moderne (Glassmorphism) avec filtres par catégorie.
- **Sécurité** : Accès par code vendeur personnalisable et protection des données via `.htaccess`.
- **Backend Léger** : Propulsé par PHP et stockage JSON (pas de base de données complexe).
